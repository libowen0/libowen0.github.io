我眼中的eminem

发表于 2017-10-16 | 分类于 心得感悟

说说那些年我们一起追的eminem

作为一个老Stan，从初中开始喜欢eminem到现在，从lose yourself 认识他，Love the way you lie 喜欢上他，再到从beautiful 学到了很多东西。有Rap god的放荡不羁，not afraid的无所畏惧，和where you gone的铁汉柔情。伴随着我走过了7个年头，伴随我走出泥泞，坚强勇敢的面对前方的未知和艰难。

eminem告诉了我一个道理，坚持做好自己就好，这是最重要的。人生路途漫漫，没有人会一帆风顺，更多的是在经受不同的考验和迷惑。时间在变，身边的人在变，唯一不会变的只有自己。唯一能够拯救自己的也只有自己的意志。

所有的病痛都会过去，所有的不如意终将消散。我们都会加油努力，为自己想要的未来而搏斗，战不罢休。

hey,guys be proud of your.

Markdown语法总结

发表于 2017-10-16 | 分类于 编程总结

Markdown，我日常使用极其频繁，不管是github提交时候的README，还是在博客园
时期的博文编写，均使用了Markdown。使用Markdown编写内容风格统一，简洁大方。
但是我每次都在使用的时候才发现有很多不会，现学现卖，导致写作效率极低。

本文旨在总结Markdown使用方法，便于快速查询相关语法。

横线

***、---、___可以显示横线效果

标题

依照不同的#个数来区分

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

文本

普通文本

我是一行普通文本

在一行开头加入一个tab或者4个空格

文本块

你好！  
我是多行文本，很高兴遇见你！  
谢谢！

在连续的每一行开头加入一个tab或者4个空格

引用

直接在行开头使用>,可实现引用

文字高亮

文字高亮能是行内部分文字高亮，使用反引号
Java MySQL maven Spring
适合做一篇文章的tag

换行

直接回车不能换行
俩个回车可以换行，但是行间距太大不够美观

直接在一行的最后多大俩个空格，在另起一行就可以换行

斜体、粗体、删除线

功能 语法
斜体 直接使用*来包围文字
粗体 直接使用**来包围文字
删除线 直接使用~~来包围文字

可以进行组合使用

代码块

使用反引号来包裹，可以直接在开头标记后添加代码类型js,会自动根据语言添加代码高亮

图片

基本格式：！[alt](URL title)alt和title即对应的HTML中的alt和title属性（都可省略）：

  • alt表示图片显示失败时的替换文本
  • title表示鼠标悬停在图片时的显示文本（主要这里要加引号）
语法 图片
![baidu](http://www.baidu.com/img/bdlogo.gif "百度logo")

链接

链接外部URL

语法 效果
[我的博客](https://libowen0.github.io/ "悬停显示") 我的博客
[我的博客][myBlog] 我的博客

图片链接

图片链接混合了图片语法和链接语法，普通链接[]内部是要显示的文字，而图片链接[]里面是要显示的图片

[![](http://www.baidu.com/img/bdlogo.gif)](http://www.baidu.com)
![][] 图片: ![alt](url title)  (alt title可省略)
[][] 链接：[name][url]

锚点

每一个标题都是一个锚点，和HTML锚点类似：

[回到标题](#标题) 回到标题

列表

无序列表

  • 可以使用*
  • 可以使用+
  • 可以使用-

多级无序列表

  • 编程语言
    • 脚本语言
      • Python

有序列表

  1. 直接使用数字+点+空格的方式
  2. 我是第二行
  3. 我是第三行

多级有序列表

  1. 家用电器
    1. 空调
      1. 美的空调

表格

单元格和表头

使用|来分割不用的单元格，使用-来分割表头和其他行

|name|age|
| -------- | --- |
| LearnShare |  12 |
| Mike       |  32 |
name age
LearnShare 12
Mike 32

对齐

在表头下方的分割线标记中加入:，即可标记下方单元格的对齐方式

  • :---代表左对齐
  • :--:代表居中对齐
  • ---:代表右对齐

diff语法

标准的markdown不支持，github markdown支持diff
版本控制的系统中都少不了diff的功能，即展示一个文件内容的增加与删除。 GFM中可以显示的展示diff效果。使用绿色表示新增，红色表示删除。

其语法与代码高亮类似，只是在三个反引号后面写diff， 并且其内容中，以 +开头表示新增，-开头表示删除。
